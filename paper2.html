6. Computer Systems

Hardware & Software

Hardware: physical components. Software: instructions executing on hardware.

System and Application Software

What is System Software?

Exam-style definition: System software enables the hardware and user applications to communicate. It manages hardware operations, provides an interface for the user, and ensures efficient execution of programs.

Utility Software

Utilities are system programs designed to perform maintenance or optimisation tasks.

  • Examples: file compression, virus scanning, defragmentation, system monitoring, backup and recovery

Exam tip: mark schemes often award one mark per correct example of a utility.

Device Drivers

Definition: Programs that allow the operating system to communicate with hardware peripherals.

Example: A printer driver translates generic OS instructions into device-specific commands.

Why drivers are needed: the OS requires a driver to translate its commands into the form the hardware understands.

Application Software

Exam-style definition: Application software is written to perform specific user-oriented tasks, rather than managing hardware.

  • General-purpose: e.g., word processors, spreadsheets
  • Special-purpose: e.g., payroll system
  • Bespoke: developed for a single organisation

Common Exam Question Styles

  • State the difference between system and application software.
  • Identify one function of an operating system.
  • Explain why a driver is needed for a new device.
  • Give two examples of utility software.

Operating System (OS) – Functions

Purpose: The OS acts as an interface between hardware and software. It hides hardware complexity from users and applications.

  1. Resource Management – controls CPU time, memory, and peripherals
  2. File Management – stores, retrieves, and organises files
  3. Process Management – multitasking and scheduling
  4. Security Management – access rights and permissions
  5. User Interface – GUI and/or CLI
  6. Device Management – communicates with hardware via device drivers

Exam tip: a valid “one function of an OS” answer could be: managing memory or providing a user interface.

Programming Languages & Translation

Software that converts source code into machine code is called a translator.

TypeFunctionExample Use
AssemblerConverts assembly language to machine codeLow-level hardware control
CompilerTranslates high-level code into an executable filePerformance and distribution (e.g., C#, Java)
InterpreterTranslates and executes code line-by-lineScripting and rapid testing (e.g., Python)
  • Know the distinction between source and object code.
  • Some languages target bytecode for execution on a virtual machine.

Exam link: assembly language requires an assembler; high-level languages are compiled or interpreted.

Boolean Algebra
