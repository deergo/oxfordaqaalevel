5. Representing Data

Number Bases

Understand decimal, binary, and hexadecimal; convert between them confidently.

Units of Information

Bit, byte, kibibyte (KiB), mebibyte (MiB), etc.; binary vs decimal prefixes.

Binary Number System

  • Unsigned and two's complement integers
  • Binary arithmetic: add, subtract, multiply
  • Fixed vs floating-point; precision, range, normalisation
  • Errors: rounding, overflow, underflow; absolute vs relative error

Characters (ASCII/Unicode)

Purpose of Unicode, code ranges, numeric vs character codes; UTF-8 awareness.

Graphics

  • Bitmap: pixel dimensions, colour depth, storage calculation
  • Vector: geometric shapes with attributes
  • Compare trade-offs

Sound

  • Sampling: rate and resolution
  • Nyquist theorem
  • File size estimation

Basic Encryption

  • Plaintext, ciphertext, key, cipher
  • Caesar and Vernam; perfect vs computational security
  • Symmetric vs asymmetric; key exchange problem

Error Detection & Correction

  • Parity bits
  • Majority voting
  • Checksums; compare effectiveness

6. Computer Systems

Hardware & Software

Hardware: physical components. Software: instructions executing on hardware.

System vs Application Software

  • System: operating systems, utilities, translators, libraries
  • Application: user-oriented programs

OS Functions

  • Scheduling, memory management
  • I/O management, interrupt handling

Programming Languages & Translation

  • Low- vs high-level languages
  • Assemblers, compilers, interpreters, JIT
  • Source vs object code; bytecode & virtual machines

7. Computer Organisation and Architecture

Internal Components

Processor, main memory, buses (address, data, control), I/O controllers; stored-program concept; von Neumann vs Harvard.

Processor Components

ALU, control unit, clock, registers (PC, CIR, MAR, MBR, SR).

Fetch–Execute Cycle

Fetch, decode, execute; interrupts and ISRs in a volatile environment.

Processor Performance Factors

  • Multiple cores, cache size/levels
  • Clock speed, word length, bus widths

Secondary Storage

Persistence; HDD vs SSD vs cloud—features, pros, cons.

Logic Gates

  • Truth tables, symbols, Boolean expressions
  • Half-adder, full-adder, D-type flip-flop

Boolean Algebra

  • Laws: commutative, associative, distributive
  • Identities, De Morgan’s laws; simplification

Instruction Format & Addressing

Opcode, operands, addressing modes (immediate, direct, indirect), mnemonics; ISA-specific.

Assembly Programming

  • Load/store, arithmetic, compare/branch, logic/shift, HALT
  • Convert between pseudocode and assembly
  • OxfordAQA instruction set (ARM-based) awareness
